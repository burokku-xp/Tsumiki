All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-04

### Added

#### コア機能
- **作業時間計測**: スタート/ストップ式タイマーで作業時間を記録
- **保存回数カウント**: ファイル保存のたびに自動カウント
- **編集ファイル追跡**: 編集したファイル名と行数をリスト化
- **変更行数**: 空行・コメント除外で変更行数を表示（参考値）
- **言語比率**: 拡張子ベースで使用言語の比率を集計
- **日次サマリー**: VSCode内サイドパネルで本日の記録を表示

#### データ管理
- **SQLiteデータベース**: ローカルにデータを永続化
- **日次データの自動リセット**: 設定した時刻に自動的に日次データをリセット
- **セッション管理**: 作業セッションの自動作成と管理
- **非アクティブセッションの自動終了**: 1時間非アクティブで自動終了

#### 表示機能
- **WebViewベースのUI**: Reactを使用したモダンなUI
- **カードベースのレイアウト**: 視覚的に分かりやすい統計表示
- **テーマカラー**: 4種類のテーマカラー（orange、blue、green、monochrome）から選択可能
- **レスポンシブレイアウト**: サイドバーの幅に応じてレイアウトが調整
- **リアルタイム更新**: ファイル保存やタイマー操作に応じて表示が自動更新

#### 設定機能
- **表示項目のカスタマイズ**: 作業時間、保存回数、ファイル数、変更行数、言語比率、ファイル一覧の表示/非表示を設定可能
- **カスタム設定UI**: 専用の設定ビューで直感的に設定を変更
- **設定の永続化**: VSCode設定に保存され、再起動後も維持

#### Slack連携
- **Incoming Webhook連携**: Slackに日次記録を投稿
- **手動投稿**: ボタンクリックで手動投稿
- **自動投稿**: 設定した時刻に自動的にSlackに投稿
- **投稿内容のカスタマイズ**: 投稿に含める項目を選択可能
- **日次コメント**: 投稿時にコメントを追加可能
- **ユーザー名設定**: Slack投稿時に表示するユーザー名を設定可能
- **リトライ機能**: 投稿失敗時に自動的にリトライ

#### 計測ルール
- **行数カウント**: 保存時のみカウント（連打防止）
- **空行・コメント除外**: 空行とコメントはカウント対象外
- **ワークスペースファイル追跡**: ワークスペース内のファイルのみを追跡
- **外部ファイル除外**: ワークスペース外のファイルは追跡対象外

### Technical Details

- **TypeScript**: strict modeで型安全性を確保
- **React**: WebView UIの構築
- **SQLite (better-sqlite3)**: データ永続化
- **esbuild**: 高速なビルドシステム
- **VSCode Extension API**: 拡張機能の基盤

### Known Limitations

- 行数は参考値として表示（空行・コメント除外のため）
- コード内容の送信は行わず、統計情報のみを記録
- プライバシー保護のため、コード内容は一切送信されません
