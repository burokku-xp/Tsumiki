# Tsumiki（積み木）

努力の積み上げを可視化するVSCode/Cursor拡張機能

## 概要

Tsumikiは、開発者の作業量を可視化し、達成感を得られる自己管理ツールです。日々の開発活動を記録・表示することで、「これだけやった」という実感を提供します。

### 主な目的

- 作業量の可視化による達成感の提供
- 日々の成長・頑張りを振り返れる開発日記

## 機能

### コア機能

- **作業時間計測**: スタート/ストップ式タイマーで作業時間を記録
- **保存回数カウント**: ファイル保存のたびに自動カウント
- **編集ファイル追跡**: 編集したファイル名と行数をリスト化
- **変更行数**: 空行・コメント除外で変更行数を表示（参考値）
- **言語比率**: 拡張子ベースで使用言語の比率を集計
- **日次サマリー**: VSCode内サイドパネルで本日の記録を表示
- **Slack投稿**: Incoming Webhook連携で日次記録をSlackに投稿
- **表示設定**: 表示項目やSlack投稿内容をカスタマイズ可能
- **テーマカラー**: 4種類のテーマカラーから選択可能

### 計測ルール

- **行数**: 保存時のみカウント（連打防止）
- **空行・コメント**: カウント対象外
- **信頼性**: 行数は参考値、作業時間・保存回数・ファイル数で総合判断

## インストール

### 方法1: .vsixファイルからインストール

1. リリースページから`.vsix`ファイルをダウンロード
2. VS Codeで `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`) を押してコマンドパレットを開く
3. `Extensions: Install from VSIX...` を選択
4. ダウンロードした`.vsix`ファイルを選択

### 方法2: ソースコードからビルド

```bash
# リポジトリをクローン
git clone <repository-url>
cd Tsumiki

# 依存関係をインストール
pnpm install

# ビルド
pnpm run build

# パッケージ化
pnpm run package
```

生成された`.vsix`ファイルを方法1と同様にインストールできます。

## 使い方

### 基本的な使い方

1. **拡張機能を有効化**: VS Codeのサイドバーに「Tsumiki」アイコンが表示されます
2. **作業時間計測を開始**: サイドパネルの「本日の記録」ビューでタイマーを開始
3. **作業を開始**: ファイルを編集・保存すると自動的に記録されます
4. **記録を確認**: サイドパネルで本日の作業量を確認できます

### 設定

設定ビューから以下の設定が可能です：

- **表示項目の選択**: 作業時間、保存回数、ファイル数、変更行数、言語比率、ファイル一覧の表示/非表示
- **Slack連携**: Webhook URLの設定、自動投稿の有効化、投稿時刻の設定
- **テーマカラー**: orange（デフォルト）、blue、green、monochromeから選択
- **日次リセット時刻**: 日次データの自動リセット時刻を設定

### Slack連携

1. SlackのIncoming Webhook URLを取得
2. 設定ビューでWebhook URLを設定
3. 自動投稿を有効化するか、手動で投稿ボタンをクリック
